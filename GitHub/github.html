Git Commands

How to push a github repo: Step by Step

  • In terminal, navigate to parent directory
  • type the following commands:
    • git status

      • This checks what files need to be added because they are modified.

    • git add .

      this will add all files with changes; you can also choose git add "filename"

    • git commit -m "meaningful commit message here"
    • git push origin
